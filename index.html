셀피 사진 선택

갤러리에서 셀피 폴더의 사진을 가능한 많이 선택해주세요. 앱이 시간순 100분위로 자동 샘플링합니다.

📂 폴더 통째로 선택 (추천)

셀카 폴더를 통째로 골라주세요. 앱이 자동으로 시간순 균등 샘플링합니다.

✨ 4천장도 OK

폴더 안의 모든 사진을 읽고 시간순으로 균등 샘플링합니다. 4천장이든 500장이든 동일하게 작동.
삼성 Z Fold: "내 파일" → DCIM/Camera 또는 셀카 폴더 → 폴더 선택

📁 또는 사진 직접 다중 선택

개별 사진들을 직접 고르고 싶을 때

📱 iOS: 길게 누른 후 드래그 🤖 Android: 길게 + 다중 선택

⚙️ 샘플링 옵션

앱이 분석할 최대 장수입니다. 시간 범위 전체에서 균등 샘플링됩니다.

⏱️ 100장 기준 약 2-4분 소요 · 200-500장 정도 골라도 충분히 풍부함

또는 데모 데이터로 체험

⚠️ 실제 사진 없이 가짜 데이터 20장으로 플로우 시연. 결과는 가짜입니다.
